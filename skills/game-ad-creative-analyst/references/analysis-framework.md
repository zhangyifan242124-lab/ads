# 数据驱动的素材分析框架

用这个框架把素材内容和账户优化动作连接起来。

## 先从卡点选择指标

优化师可以直接说业务问题，不需要指定指标。先把问题翻译成诊断方向，再从数据表里找证据。

| 优化师卡点 | 优先指标 | 辅助指标 | 主要判断 |
|---|---|---|---|
| CPI 太高，想降下来 | CTR、CVR、CPI、spend、installs | CPM、CPC、region、placement | 是没人点、点了不装、流量太贵，还是样本不足 |
| CTR 不差但装不动 | CVR、CPI、clicks、installs | 商店页转化、素材承接、CTA | 是素材预期和安装承接断裂，还是商店页/包体问题 |
| CTR 低，想提高吸引力 | CTR、impressions、clicks | CPM、CPC、首帧、前 3 秒 | 是开头弱、卖点不清，还是平台流量不匹配 |
| ROAS 差，怀疑骗量 | ROAS、retention、ARPU、pay rate | CTR、CVR、CPI、素材误导点 | 是低质量吸量、玩法错配，还是付费链路弱 |
| 素材能不能放量 | spend、CPI、ROAS、retention、test_duration | CPM、CTR trend、CVR trend | 吸引、成本、质量是否同时过线 |
| 素材最近跑不动了 | 分日 CTR、CPI、CPM、frequency、spend trend | 历史同素材表现、预算变化 | 是素材疲劳、竞价变贵，还是账户调整导致 |
| 想判断是否停投 | spend、installs、CPI、ROAS、retention | CTR、CVR、审核风险 | 样本是否足够弱，是否还有可迭代价值 |
| 想找下一版方向 | 当前素材表现、历史成功/失败素材 | 视频结构、卖点、平台风格 | 保留高贡献段落，拆变量做下一批 brief |

如果用户给了很多指标，先输出“本次核心依据”和“暂不作为核心依据”的区分。不要逐项解释所有字段。

如果关键指标缺失，继续分析，但要说明结论边界。例如没有留存和 ROAS 时，只能判断前端买量表现，不能判断用户质量和放量价值。

## 单条和多条素材模式

### 单条素材深挖

目标是回答：这条素材为什么这样表现，是否继续投，下一轮应该改哪里。

优先检查：

- 素材变量：首帧、前 3 秒钩子、核心卖点、视频结构、画面信息量、文案话术、CTA、落地承接。
- 节奏校验：开头、中段、结尾。
- 分日趋势：CTR、CVR、CPI/CPA、CPM/CPC、消耗、频次是否稳定。
- 视频观看漏斗：3秒、25%、50%、75%、99%/完播。
- 后链路质量：留存、ROAS、付费率、ARPU/ARPPU。
- 素材承诺和真实玩法是否一致。

输出重点：

- 单条素材类型判断。
- 主要问题发生在哪一段。
- 继续投、改后复测、暂停的账户动作。
- 简略迭代方向。

### 多条素材横评

目标是回答：这一批里哪个方向值得继续做，哪些变量有效，哪些素材该停。

优先检查：

- 视频和数据是否能按素材 ID 匹配。
- 是否为同平台、同地区、同周期、同目标；如果不是，标注比较边界。
- 每条素材的关键变量：首帧、前 3 秒钩子、核心卖点、视频结构、画面信息量、文案话术、CTA、落地承接。
- 数据分层：前端吸量、转化承接、后链路质量、成本、疲劳。

输出重点：

- 每条素材必须先完成基础创意拆解：首帧、前 3 秒钩子、核心卖点、视频结构、画面信息量、文案话术、CTA、落地承接、优点、问题。
- 素材排序和分层，而不是逐条长篇。
- 高表现素材的共同变量。
- 跑不出来素材的共同问题。
- 下一轮测试矩阵：保留变量、替换变量、停止变量、新增变量。

多条素材时只深挖三类代表：

- 最值得继续投或小幅放量的 1-3 条。
- 数据一般但有学习价值的 1-3 条。
- 明确建议停的素材类型。

不要只看数据下结论。必须把视频变量和数据表现绑定起来，例如：

```text
XKT045 的 CTR 高，可能来自角色 UI + 三选一的强信息差；但留存/ROAS 为 0，说明这个钩子可能吸引了好奇点击，而没有筛出高质量用户。
```

## 素材变量拆解字段

默认用这些字段拆视频，三段式只用于辅助判断节奏：

| 字段 | 要回答的问题 | 主要对应数据 |
|---|---|---|
| 首帧 | 第一眼看到什么，能否立刻停住 | 3秒播放率、CTR |
| 前3秒钩子 | 靠什么吸引停留：冲突、结果、挑战、福利、对比、强爽感 | 3秒播放率、CTR |
| 核心卖点 | 讲的是爽感、福利、角色、玩法、剧情还是数值成长 | CVR、留存、ROAS |
| 视频结构 | 先给结果、先制造冲突、先教学、直接展示玩法、还是对比 | 25%/50%播放率、CVR |
| 画面信息量 | 一眼能不能看懂，是否过乱或过空 | CTR、播放漏斗 |
| 文案话术 | 夸张、疑问、教程、挑战、福利、对比、命令式 CTA | CTR、CVR、审核风险 |
| CTA | 是否明确引导点击、下载、试玩或继续 | CVR、CPI/CPA |
| 落地承接 | 点击后看到的内容是否和素材承诺一致 | CVR、留存、ROAS |

数据解释时按这个映射：

- CTR/3秒播放主要看首帧、前3秒钩子、文案话术和画面信息量。
- CVR/CPI 主要看核心卖点、视频结构、CTA 和落地承接。
- 留存/ROAS 主要看核心卖点是否贴合真实玩法、素材承诺是否过度、用户质量是否被筛选。
- 播放中段掉点主要看视频结构、信息递进和画面变化。

## 报表读取顺序

分析 CSV 或表格时先做这几步：

1. 识别字段：素材名、计划名、平台、地区、消耗、曝光、点击、安装、CTR、CVR、CPI、CPM、CPC、留存、ROAS、ARPU。
2. 标记数据层级：单条素材、广告组、计划、地区汇总、总汇总。
3. 排除不适合直接比较的行：汇总行、空素材行、样本极小行。
4. 如果平台或地区只出现在命名里，可以推断，但必须说明“根据命名推断”。
5. 同一素材多行时，优先按平台/地区/计划拆开比较，再判断整体。
6. 多条素材时，先生成匹配表：素材 ID、视频文件、数据行、匹配依据、匹配置信度。

不要把汇总行和单素材行放在一起比较，否则会误判素材表现。

## 字段充足不等于分析充足

如果用户已经提供单条素材的消耗、曝光、点击、安装、CTR、CVR、CPI、留存、ROAS、频次等字段，不要再把这些字段列为“还需要的数据”。

这时要区分两类缺口：

- **字段缺口**：当前素材缺少某些必要指标，例如没有留存或没有 ROAS。
- **对照缺口**：当前素材字段完整，但缺少同批素材、分日趋势、变量版本、版位、国家拆分或历史基准。

测试期素材复盘通常缺的是对照维度，不是单条素材字段。表达时要说清楚：

```text
这条素材的核心字段已经够做单条判断；如果要解释为什么跑成这样、下一轮该拆哪些变量，还需要同批对照或分日趋势。
```

## 核心分类

给素材优先归为一个主类型：

- **吸量型**：CTR 强，CPI 可能低，但 CVR、留存、ROAS 不一定证明用户质量好。
- **转化型**：CTR 一般或偏低，但 CVR 和后链路质量好。
- **骗点型**：CTR 强，但 CVR、留存或 ROAS 弱，常见原因是预期落差。
- **精准小众型**：量不大，但质量高，适合控量或特定人群。
- **疲劳型**：历史赢家素材开始 CTR 下降、CPI 上升或消耗能力下降。
- **承接断裂型**：开头吸引人，但中段、结尾、商店页或真实玩法承接不上。
- **平台错配型**：素材逻辑可能适合某个平台，但不适合当前平台节奏、原生感或审核尺度。

## 常见数据组合

### CTR 高，CVR 低

可能原因：

- 钩子吸引人，但有误导。
- 点击前承诺强，点击后玩法承接弱。
- CTA 或商店页与素材预期不一致。
- 用户只是好奇，不是真正有安装意愿。

建议动作：

- 如果审核安全，保留钩子。
- 重做中段真实玩法证明。
- 在 CTA 前更早露出游戏真实身份。
- 测一版降低夸张程度的素材。

### CTR 低，CVR 高

可能原因：

- 前 3 秒不够抓人。
- 愿意点击的人很精准。
- 核心卖点出现太晚。

建议动作：

- 重做前 3 秒。
- 保留核心玩法和承诺。
- 测更强的视觉反差、失败开局或更快的奖励反馈。

### CPI 低，留存或 ROAS 弱

可能原因：

- 吸引到泛用户，但质量低。
- 包装误导。
- 人群和真实玩法错配。
- 激励、提现、奖励表达过强。

建议动作：

- 不要直接扩量。
- 收紧素材承诺。
- 增加真实玩法占比。
- 先比较 cohort 质量再决定是否放大。

### CTR 低，CPI 高，质量未知

可能原因：

- 钩子弱。
- 玩法不清楚。
- 第一帧差。
- 平台风格不匹配。

建议动作：

- 样本足够时暂停。
- 样本不足时换开头复测。
- 没有足够安装数前，不要判断用户质量。

### CTR 稳定，CPI 上升

可能原因：

- 竞价环境变贵。
- 人群饱和。
- 素材疲劳还没完全体现在 CTR 上。
- 平台学习波动。

建议动作：

- 查看 CPM 和频次。
- 优先刷新开头，不一定推翻整条素材。
- 如果数据支持，按地区或人群拆分观察。

## 账户优化视角

必须把诊断翻译成账户动作：

- **继续测试**：趋势有潜力，但证据还不成熟。
- **谨慎放量**：吸引力和质量都能接受，但仍需观察。
- **迭代复测**：概念可用，但某个段落拖累表现。
- **拆 A/B 测试**：多个原因都合理，需要用变量测试拆开。
- **暂停**：样本足够且表现弱，或审核/账户风险明显。
- **补数据后判断**：样本太小或关键指标缺失。

## 证据等级

输出信心等级：

- **高**：数据一致，样本量有意义，关键指标完整。
- **中**：趋势明确，但缺少一个关键指标。
- **低**：样本小、数据不完整，或只有素材没有投放数据。

如果只有视频，只分析创意潜力和审核风险，不要把表现推断成事实。
